# Vite React Cloudflare

Use this general-purpose stack for interactive applications, dashboards, tools,
and products that benefit from a client-rendered React UI plus Cloudflare Worker
routes or Durable Object persistence.

Built with React, Vite, Tailwind CSS, shadcn/ui configuration, Hono, and one
Cloudflare Durable Object. Prefer `next-cf` when server rendering or SEO is a
primary requirement.
